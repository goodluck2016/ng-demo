## ng-demo
a project of ng

-------------------------------------------------------------------------------
#### ng-cli搭建

- step1:
`cnpm install -g @angular/cli`
检验成功与否：ng -v

- step2:
`ng new ng-demo`

- step3:
`cd ng-demo`

- step4:
`ng serve`
